# Caça ao Tesouro

## Descrição

Jogo desenvolvido em linguagem C para a unidade curricular.

## Funcionalidades

- Menu principal
- Movimento do jogador
- Mapa 10x10
- Chave
- Tesouro
- Armadilhas
- Sistema de vidas
- Sistema de pontuação
- Gravação de pontuações em ficheiro

## Compilação

```bash
gcc main.c jogo.c -o jogo
```

## Execução

```bash
./jogo
```
